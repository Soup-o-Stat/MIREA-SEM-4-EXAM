# Ответы на теоретические вопросы по билетам

## Билет №1

### 1. Управленческие решения и их классификация. Общий случай задачи принятия решений. Процесс и методы принятия решений.

**Управленческие решения** - это выбор альтернативы, осуществляемый руководителем в рамках его должностных полномочий.

**Классификация:**
- По степени структурированности: программируемые и непрограммируемые
- По характеру целей: стратегические, тактические, оперативные
- По методам обоснования: интуитивные, адаптивные, рациональные

**Общий случай задачи принятия решений** включает:
1. Множество альтернатив
2. Критерии оценки
3. Ограничения
4. Цель - выбор оптимальной альтернативы

**Процесс принятия решений:**
1. Идентификация проблемы
2. Сбор информации
3. Формулировка критериев
4. Генерация альтернатив
5. Оценка альтернатив
6. Выбор решения
7. Реализация и контроль

**Методы:**
- Экспертные (мозговой штурм, Дельфи)
- Математические (линейное программирование)
- Многокритериальные (Парето, ELECTRE)

### 2. Прямая и двойственная задачи. Симметричные или несимметричные задачи. Общие правила составления двойственных задач.

**Прямая задача** - исходная задача оптимизации.

**Двойственная задача** - связанная задача, где ограничения становятся переменными.

**Симметричные задачи** имеют одинаковую структуру (напр., максимизация с "≤" → минимизация с "≥").

**Несимметричные** - разную структуру (наличие равенств).

**Правила составления:**
1. Ограничениям прямой соответствуют переменные двойственной
2. Коэффициенты цели → правые части ограничений
3. Матрица коэффициентов транспонируется
4. Максимум ↔ минимум

## Билет №2

### 1. Термины и понятия многокритериальной оптимизации. Лицо, принимающее решение. Альтернатива. Критерии. Исход. Задача многокритериальной оптимизации. Векторные и скалярные методы решения.

**ЛПР** - субъект, выбирающий решение.

**Альтернатива** - вариант решения.

**Критерии** - показатели оценки.

**Исход** - результат выбора.

**Задача** - найти альтернативу, оптимальную по нескольким критериям.

**Векторные методы:**
- Парето (недоминируемые решения)
- ELECTRE (граф предпочтений)

**Скалярные методы:**
- Взвешенные суммы
- Идеальная точка

### 2. Первая теорема двойственности. Алгоритм решения двойственной задачи через обратную матрицу.

**Первая теорема:** если обе задачи имеют допустимые решения, то они имеют оптимальные решения с равными значениями целевых функций.

**Алгоритм:**
1. Записать прямую задачу
2. Построить двойственную
3. Решить прямую симплекс-методом
4. Найти обратную матрицу для базисных переменных
5. Оптимальные двойственные переменные = коэффициенты цели × обратную матрицу

## Билет №3

### 1. Парето-оптимальность. Векторная оценка. Графическая интерпретация. Методы сужения множества Парето. Проблемы решения задач методом Парето.

**Парето-оптимальность** - нельзя улучшить один критерий без ухудшения другого.

**Векторная оценка** - вектор значений критериев.

**Графически** - граница множества допустимых решений.

**Методы сужения:**
- Взвешенные суммы
- Идеальная точка
- ELECTRE

**Проблемы:**
- Много недоминируемых решений
- Субъективность выбора

### 2. Вторая теорема двойственности. Условия дополняющей нежесткости с точек зрения рентабельности и дефицитности.

**Вторая теорема:** для оптимальных решений выполняются условия:
- Если переменная > 0, соответствующее ограничение = равенство
- Если ограничение строгое, переменная = 0

**Рентабельность:** ресурс используется полностью, если оценка > 0.

**Дефицитность:** если ресурс не используется полностью, оценка = 0.

**Этапы ELECTRA:**
1. Формирование матрицы решений
2. Нормализация
3. Построение матрицы согласия и несогласия
4. Определение отношений предпочтения
5. Построение графа

**Гипотеза превосходства:** альтернатива A лучше B, если:
- Согласие ≥ порога
- Несогласие ≤ порога

**Индекс несогласия** - мера невыгодности.

**Матрица предпочтений** показывает силу предпочтения.

**Граф** визуализирует отношения.

**Увеличение порога** делает критерии строже.

### 2. Третья теорема двойственности. Нижняя и верхняя границы интервала устойчивости двойственных оценок.

**Третья теорема:** двойственные оценки показывают, как изменится целевая функция при изменении ресурсов.

**Границы устойчивости** - диапазон, в котором оценки остаются valid.

## Билет №7

### 1. Метод Анализа Иерархий. Устранение комплексной несогласованности. Методики расчетов Индекса Согласованности (ИС) и Отношения Согласованности (ОС).

**ИС = (λmax - n)/(n - 1)**

**ОС = ИС/СИ**

где СИ - случайный индекс.

**Устранение:** если ОС > 0.1, пересмотреть оценки.

### 2. Понятие выпуклого множества в линейном программировании. Точки выпуклого множества. Основная теорема линейного программирования.

**Точки:**
- Внутренние
- Граничные
- Угловые (вершины)

**Основная теорема:** если решение существует, оно достигается в угловой точке.

**Стандартная форма:**
- Максимизация
- Ограничения "≤"

**Матричная запись:**

при Ax ≤ b

4. Свободные переменные → разность двух неотрицательных
